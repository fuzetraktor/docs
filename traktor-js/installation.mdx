A instalação do **Traktor.js** é simples e pode ser feita através do <Tooltip tip="Google Tag Manager">GTM</Tooltip>
para garantir fácil gerenciamento e aplicação em todas as páginas do site.

## Passos para Instalação

Acesse o [Google Tag Manager](https://tagmanager.google.com/)

### Crie uma nova tag

**Nome:** Traktor.js

**Tipo de tag:** HTML Personalizado

```
<script src="https://js.gotraktor.com/traktor.js"></script>
```

<Tip>
  Configure o acionamento da tag para que ela seja disparada em todas as
  páginas.
</Tip>

### Salve e publique a tag

Com isso, o script será carregado em todas as páginas do seu site, começando a coletar informações de origem de tráfego de forma automática.
