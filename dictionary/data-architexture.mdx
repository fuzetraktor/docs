## Data Structure

A estrutura de dados refere-se à organização e armazenamento eficiente de dados, facilitando o acesso, a modificação e a manipulação. As estruturas de dados são fundamentais para o desenvolvimento de sistemas de rastreamento, CRM e plataformas de marketing digital.

## Data Model

O modelo de dados define a estrutura e as relações entre os dados armazenados, como dados de leads, campanhas e métricas. Ele ajuda a garantir que as informações sejam armazenadas de forma consistente e acessível, promovendo integridade e eficiência.

## Tracking Tags

São pequenos pedaços de código inseridos nas páginas web para coletar dados de usuários, como comportamentos e interações. Utilizadas para associar visitantes e leads com campanhas e origens de tráfego.

## Cookies

Arquivos pequenos armazenados no navegador que contêm informações sobre o comportamento de navegação do usuário, como UTM parameters, fonte de tráfego e campanhas. Eles são usados para rastrear a jornada do usuário entre diferentes visitas e dispositivos.

## Cookie Expiry

O tempo de validade dos cookies que armazena os dados de origem e comportamento do usuário. Após o tempo de expiração, os dados são deletados, e o histórico do usuário é resetado, impactando a rastreabilidade.
